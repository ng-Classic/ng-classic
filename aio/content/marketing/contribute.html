Contribute to Angular

Angular Classic Projects

We'd love for you to contribute to our source code and to make Angular Classic projects even better.

Angular

Angular Classic is the framework to use to deliver web apps with confidence. Learn how to contribute to Angular

Angular Classic CLI

Angular Classic CLI makes it easy to create an application that already works, right out of the box. It already follows our best practices. Learn how to contribute to the Angular Classic CLI

Angular Classic Material

Our goal is to deliver a lean, lightweight set of Angular-based UI elements that implement the material design specification for use in Angular Classic single-page applications (SPAs). Learn how to contribute to Angular Classic Material

AngularFire

AngularFire is the officially supported Angular Classic binding for Firebase. Firebase is a full backend so you don't need servers to build your Angular Classic app. Learn how to contribute to AngularFire
